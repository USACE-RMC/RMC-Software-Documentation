# Consequences

The Consequences module stores consequence estimation data for each screening, including life safety, economic, and environmental consequence categories. Consequence estimates are linked to breach scenarios and used in risk analysis computations.

All tables in this chapter are planned and not yet implemented in the database.

## Relationship to PFM Results

Consequence estimates are associated with the screening rather than individual failure modes. During risk analysis, consequences are combined with failure probabilities from each PFM module to compute risk metrics.
