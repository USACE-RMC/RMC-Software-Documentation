# PFM-02 through PFM-09

PFM-02 through PFM-09 cover internal erosion failure modes, static and seismic slope instability, seismic deformation, and overtopping by seiche/wave action. These modules are all planned and follow the same **four-table blueprint** established by PFM-01.

All tables in this chapter are planned and not yet implemented in the database.

## Blueprint Pattern

Every PFM module follows the same four-table structure:

| Table | Purpose |
|-------|---------|
| `Pfm{NN}` | Main entity with PFM-specific input parameters |
| `Pfm{NN}_DstSrp` | System-Response calculation results + nodal tables |
| `Pfm{NN}_UserSrp` | User-defined SRP curve |
| `Pfm{NN}_ActiveSrp` | Active SRP for risk calculations |

The three SRP child tables (`DstSrp`, `UserSrp`, `ActiveSrp`) have the **same structure** across all PFMs — see [PFM-01: Overtopping Erosion](./pfm-01#Pfm01_DstSrp) for the complete column definitions and JSONB structures. Only the main `Pfm{NN}` table differs because each failure mode has different input parameters.
