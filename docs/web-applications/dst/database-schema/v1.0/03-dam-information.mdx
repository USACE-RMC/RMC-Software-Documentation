# Dam Information

The Dam Information module stores all metadata about the dam being screened, including general identification, reservoir characteristics, spillway data, embankment geometry, and material properties. Frontend pages for dam information exist in the DST application, but the database tables have not yet been implemented.

All tables in this chapter have `status="planned"` — column definitions are representative placeholders based on anticipated application needs.

## Data Sources

Dam information can be populated from two sources:

1. **National Inventory of Dams (NID)** — Automatic import using the NID ID, which populates general identification, location, and basic engineering data.
2. **Manual entry** — Users enter or override values directly in the DST application for detailed geometry, materials, and cross-section data.
