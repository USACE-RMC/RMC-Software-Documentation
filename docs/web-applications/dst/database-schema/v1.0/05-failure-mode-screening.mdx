# Failure Mode Screening & Selection

The Failure Mode Screening and Selection module manages which potential failure modes (PFMs) are applicable to a given dam and which ones the user has selected for detailed evaluation. This serves as the gating step between dam characterization and failure mode analysis.

All tables in this chapter are planned and not yet implemented in the database.

## Workflow

The failure mode screening and selection process follows this workflow:

1. **Screening step** — The user reviews all 33 potential failure modes and marks each as applicable or not applicable to the dam, providing justification.
2. **Selection step** — From the applicable failure modes, the user selects which ones to evaluate in detail and assigns priority rankings.
3. **Evaluation** — Selected failure modes are then evaluated using their respective PFM module (Chapters 6-8).
