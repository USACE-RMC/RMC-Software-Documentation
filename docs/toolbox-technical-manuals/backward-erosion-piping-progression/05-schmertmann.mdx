# Schmertmann

Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/> developed an approach for estimating the factor of safety
(_FS_) regarding piping using teh concept of ambient, pre-pipe, hydraulic gradients along the potential piping path based on laboratory flume tests. The FS for
pipe progression is given in <EquationReference parentDocId="108" equationKey="fs-pipe-progression" />:

<Equation parentDocId="108" equationKey="fs-pipe-progression" equation="FS = \left(\frac{C_D C_L C_S C_K C_γ C_Z C_α i_{\textit{pmt}}}{C_R i_f}\right)" />

where:

> _i<sub>pmt</sub>_ = maximum pre-pipe hydraulic gradient along the pipe path in the reference laboratory test (that is, laboratory horizontal critical gradient)  
> _i<sub>f</sub>_ = maximum pre-pipe hydraulic gradient along the pipe path in the field  
> _C<sub>D</sub>_ = depth/length factor  
> _C<sub>L</sub>_ = total pipe length factor  
> _C<sub>S</sub>_ = grain-size factor  
> _C<sub>K</sub>_ = anisotropic permeability factor  
> _C<sub>γ</sub>_ = density factor  
> _C<sub>Z</sub>_ = underlayer factor  
> _C<sub>α</sub>_ = adjustment for pipe inclination  
> _C<sub>R</sub>_ = embankment axis curvature factor

Schmertmann applied these correction factors to more than 100 laboratory flume tests to adjust the laboratory results to the reference values shown
in <TableReference parentDocId="108" tableKey="schmertmann-reference-values"/>.

From the corrected values, Schmertmann proposed a no-test default relationship as the default value of _i<sub>pmt</sub>_ to use in assessments if
laboratory flume tests were not conducted, shown in <EquationReference parentDocId="108" equationKey="i-pmt" />
and <FigReference parentDocId="108" figKey="schmertmann-lab-critical-gradients"/>.

<Equation parentDocId="108" equationKey="i-pmt" equation="i_{pmt} = 0.05 + 0.183(C_u - 1)" />

where

> <i>C<sub>u</sub></i> = coefficient of uniformity

Equations 1 and 2 can be used along with the correction factors to assess the FS for BEP progression in the field. While this figure is quite useful,
it is difficult to estimate the uncertainty in the critical point gradient because study averages were presented. For example, one of the single data
points is the average from 14 individual laboratory tests.

To assess the likelihood of BEP progression, Robbins and Sharp <Citation citationKey="Robbins2016" bibFile="/bibliographies/108-bib.json"/> compiled
all the data in Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/> and developed a probabilistic chart for
determining ipmt, again relative to the reference values, as shown in <FigReference parentDocId="108" figKey="schmertmann-lab-probabilities"/>.
Robbins and O’Leary <Citation citationKey="Robbins2020" bibFile="/bibliographies/108-bib.json"/> expanded this chart to include 0 percent and 100
percent probability trend lines based on linear extrapolation of the slopes and intercepts of the quantile regression lines.

To calculate a probability of BEP progression using the lines shown on the probability chart, the value of _i<sub>pmt</sub>_ must be calculated
for a given field scenario as shown in Equation 3.

$$
FS = \frac{{C_R}{i_f}} {{C_D}{C_L}{C_S}{C_K}{C_γ}{C_Z}{C_α}} \tag{3}
$$

The probabilistic chart from Robbins and Sharp <Citation citationKey="Robbins2016" bibFile="/bibliographies/108-bib.json"/> shows that the uncertainty
with the laboratory critical gradient increases with increasing _C<sub>u</sub>_, where there is less data. In addition, at higher _C<sub>u</sub>_ values,
soils tend to become internally unstable. Therefore, the probabilistic chart is limited to a _C<sub>u</sub>_ between 1.1 and about 4.

## Piping Layer Characterization

Step 2 characterizes the piping layer. The input includes the depth (thickness) of the piping layer (_D_), coefficient of uniformity (_Cu_), grain size
in the field with 10 percent finer by weight (_d₁₀f_), anisotropy in the field of the piping layer (_Rkf_), relative density in the field (_Drf_), and
angle of the pipe path (_α_). The adjusted Schmertmann method uses the probabilistic chart of Robbins and Sharp <Citation citationKey="Robbins2016" bibFile="/bibliographies/108-bib.json"/>,
expanded by Robbins and O’Leary <Citation citationKey="Robbins2020" bibFile="/bibliographies/108-bib.json"/>, for both the deterministic and probabilistic
methods and is limited to _Cu_ between 1.1 and 4. _Cu_ values outside of this range have an orange background. The depth (thickness of the piping layer)
is in the direction perpendicular to the angle of the pipe path discussed later in this section.

The selections in Step 1 affect the input for Step 2, and cells that do not apply have a gray background. These cells are not used in subsequent
calculations even if data is present.

For deterministic analysis, input only the most likely values. The mean value used for subsequent calculations is the most likely (or mode)
value. <FigReference parentDocId="108" figKey="schmertmann-step2-deterministic"/> illustrates the deterministic input.

The remaining input for step 2 addresses pipe path geometry as illustrated in <FigReference parentDocId="108" figKey="schmertmann-pipe-path"/> and
includes the angle of the pipe path (_α_) with respect to horizontal and the direct (not meandered) length between ends of a completed pipe path (_L_).
A figure is included in this step as a guide.

In most cases, the pipe path is horizontal, and _α_ = 0 degrees. The angle is positive for pipe paths that progress upwards and negative for pipe paths
that progress downwards. Based on Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/>, the range for _α_ is
between -90 degrees and 40 degrees. Cells where the input parameter is outside these limits have an orange background.

Based on the user-specified anisotropy, the ratio of the depth of the piping layer to the length of the pipe path in the field (_D/Lf_) is calculated,
and the transformed length of the piping path in the field (_Lf_) is calculated using Equation 4.

$$
L_f = \frac{L} {(R_{kf})^{0.5}} \tag{4}
$$

where:

> _R<sub>kf</sub>_ = anisotropy in the field of the piping layer (_k<sub>h</sub>/k<sub>v</sub>_)

## Laboratory Horizontal Critical gradient

Step 3 estimates the laboratory horizontal critical gradient. Use the drop-down list to indicate if a laboratory flume test was performed. Cells that
do not apply have a gray background. For a laboratory flume test, input the measured laboratory horizontal critical gradient as illustrated
in <FigReference parentDocId="108" figKey="schmertmann-flume-gradient"/>. In practice, laboratory flume tests are rarely conducted to measure the critical
horizontal gradient, and the linear relationship proposed by Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/> is
the primary means for estimating critical gradient values as a function of coefficient of uniformity (_C<sub>u</sub>_). In this toolbox, the 50 percent quantile
line from Robbins and Sharp <Citation citationKey="Robbins2016" bibFile="/bibliographies/108-bib.json"/> is used in lieu of Schmertmann’s no-test default
line using Equation 5 as illustrated in <FigReference parentDocId="108" figKey="schmertmann-lab-gradient"/>.

$$
i_{pmt} = 0.1358(C_u) + 0.002 \tag{5}
$$

Robbins and Sharp found that the best fit, median line of all test results indicates Schmertmann's no-test default line is conservative for _C<sub>u</sub>_ < 3
and unconservative for _C<sub>u</sub>_ > 3.

## Field Horizontal Critical Gradient

Step 4 calculates the field horizontal critical gradient by applying the various correction factors to the estimated or measured laboratory horizontal critical
gradient as illustrated in <FigReference parentDocId="108" figKey="schmertmann-field-corrections"/>. The correction factors for foundation depth, seepage path
length, grain size, anisotropy, and soil density are estimated using Equations 6, 7, 8, 9, and 10. Since a straight axis is assumed for the embankment, there
is no correction factor for curvature in the embankment axis, and _C<sub>R</sub>_ = 1.

$$
C_D = \frac{\left(\frac{D}{L_f}\right)^{\frac{0.2}{\left(\frac{D}{L_f}\right)^2-1}}}{\left(\frac{1 ft}{5 ft}\right)^{\frac{0.2}{\left(\frac{1 ft}{5 ft}\right)^2-1}}} = \frac{\left(\frac{D}{L_f}\right)^{\frac{0.2}{\left(\frac{D}{L_f}\right)^2-1}}}{1.4} \tag{6}
$$

$$
C_L = \left(\frac{5 ft}{L_f}\right)^{0.2} \tag{7}
$$

$$
C_S = \left(\frac{d_{10f}}{0.20 mm}\right)^{0.2} \tag{8}
$$

$$
C_K = \left(\frac{1.5}{\frac{k_h}{k_v}}\right)^{0.5} = \left(\frac{1.5}{R_{kf}}\right)^{0.5} \tag{9}
$$

$$
C_γ = 1 + 0.4\left(\frac{D_{rf}}{100}-0.6\right) \tag{10}
$$

where:

> _D/L<sub>f</sub>_ = ratio of depth of piping layer to length of pipe path in the field  
> _L<sub>f</sub>_ = transformed length (feet) of the pipe path in the field  
> _k<sub>h</sub>/k<sub>v</sub>_ = anisotropy in the field (_R<sub>kf</sub>_)  
> _D<sub>rf</sub>_ = relative density in the field

<Figure

figKey="schmertmann-field-corrections"

A higher permeability layer beneath the piping layer increases the flow toward the pipe and decreases the critical gradient. Use the drop-down list to indicate
the presence of a higher permeability underlayer (for example, fine sand layer over coarse sand layer). If there is no higher permeability underlayer, _C<sub>Z</sub>_ = 1
as illustrated in <FigReference parentDocId="108" figKey="schmertmann-no-underlayer"/>. Cells that do not apply have a gray background. If a higher permeability
underlayer exists, input the permeability of the piping layer (_k<sub>p</sub>_), permeability of the underlayer (_k<sub>u</sub>_), and equivalent radius of the
developing pipe cross section (_r_) to estimate the underlayer correction factor (_C<sub>Z</sub>_).

Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/> provided _C<sub>Z</sub>_ as a function of _D∕r_ for _k<sub>u</sub>∕k<sub>p</sub>_
≤ 1, _k<sub>u</sub>∕k<sub>p</sub>_ = 5, and _k<sub>u</sub>∕k<sub>p</sub>_ ≥ 1 based on very limited data derived from flow nets of the University of Florida
flume. CZ is obtained from two-way linear interpolation using these three curves as illustrated in <FigReference parentDocId="108" figKey="schmertmann-underlayer"/>.
Fell et al. <Citation citationKey="Fell2008" bibFile="/bibliographies/108-bib.json"/> suggests using _C<sub>Z</sub>_ = 1 since r is very small and _D∕r_ is typically
very large in practice; _C<sub>Z</sub>_ is approximately 1 when _D∕r_ is greater than about 80. For very thin erodible layers (less than a couple of feet),
Fell et al. <Citation citationKey="Fell2008" bibFile="/bibliographies/108-bib.json"/> suggested using r = 2.5 to 10 mm, where the horizontal critical gradient
decreases as r increases. If thin alternating layers of erodible and non-erodible soil are modeled as a homogenous layer with high anisotropy, use _C<sub>Z</sub>_ = 1.

$$

## Pipe Path Inclination

Step 5 determines the adjustment for pipe path inclination. In most cases, the pipe path is horizontal (_α_ = 0 degrees), and _C<sub>α</sub>_= 1. Cells that do not apply
have a gray background. If the pipe path inclination is not zero, Schmertmann <Citation citationKey="Schmertmann2000" bibFile="/bibliographies/108-bib.json"/> provided
the field critical gradient adjusted for pipe path inclination (_i<sub>pα</sub>_) as a function of _α_ for field horizontal critical gradients (_i<sub>po</sub>_) of 0.05,
0.10, 0.15, 0.20, 0.25, 0.30, 0.35, 0.40, 0.45, and 0.50 or greater, as illustrated in <FigReference parentDocId="108" figKey="schmertmann-pipe-path-inclination"/>. _i<sub>pα</sub>_ is
obtained from two-way linear interpolation using these ten curves. The back-calculated adjustment for pipe path inclination uses Equation 12.

$$
C_α = \frac{i_{pα}}{i_{po}} \tag{12}
$$

## Field Critical Gradient for BEP progression

Step 6 adjusts the field horizontal critical gradient for pipe path inclination using the adjustment for pipe path inclination (_C<sub>α</sub>_) from step 4 using Equation 13.

$$
i_{pα} = i_{ch} C_α \tag{13}
$$

If the pipe path is horizontal, _i<sub>pα</sub>_ = _i<sub>ch</sub>_ because _C<sub>α</sub>_ = 1.

Schmertmann’s adjusted model applies to 2D seepage with plane or ditch exits parallel to the embankment centerline. Van Beek et al. <Citation citationKey="vanBeek2015" bibFile="/bibliographies/108-bib.json"/> found
that three-dimensional (3D) configurations with flow toward a single point (such as a hole in a confining layer) resulted in significantly smaller critical gradients than
predicted by the Dutch piping model. In both the small and medium-scale experiments, the model overestimated the critical gradient by a factor of approximately two. Although
not developed and calibrated for this method, a similar reduction for a 3D exit condition is appropriate. If the field horizontal critical gradient is further adjusted for
3D flow, input a user-specified gradient reduction factor (GRF). Cells that do not apply have a gray background. <FigReference parentDocId="108" figKey="schmertmann-gradient-reduction-factor"/>
and <FigReference parentDocId="108" figKey="schmertmann-gradient-no-reduction-factor"/> illustrate the two scenarios.

## Likelihood of Backward Erosion Piping Progression

Robbins and Sharp <Citation citationKey="Robbins2016" bibFile="/bibliographies/108-bib.json"/> presented the results of a best-fit quantile regression analysis
from the individual laboratory flume tests, which were expanded by Robbins and O'Leary <Citation citationKey="Robbins2020" bibFile="/bibliographies/108-bib.json"/>
to include 0 percent and 100 percent probability trend lines based on linear extrapolation of the slopes and intercepts of the quantile regression lines. Step 7
uses this modified chart to develop a cumulative density function to estimate the probability of BEP progression as illustrated
in <FigReference parentDocId="108" figKey="schmertmann-cdf"/>.

<Figure 
parentDocId="108" 
figKey="schmertmann-cdf" 
src="figures/toolbox-technical-manuals/backward-erosion-piping-progression/figure27.png" 
alt="Step 7 of Schmertmann worksheet: Cumulative density function for backward erosion piping progression."
caption="Step 7 of Schmertmann worksheet: Cumulative density function for backward erosion piping progression."></Figure>

The average horizontal gradient in the foundation at the pipe head is calculated by dividing the net hydraulic head by the seepage path length using Equation 14.

$$
i_{avf} = \frac{ΔH}{L} = \frac{HW-TW}{L} \tag{14}
$$

where:

> _ΔH_ = net hydraulic head (feet)
> _HW_ = headwater level (feet)
> _TW_ = tailwater level (feet)
> _L_ = direct (not meandered) seepage path length (feet)

The FS against BEP progression is calculated using Equation 15.

$$
FS = \frac{i_{pα}}{i_{avg}} \tag{15}
$$

where:

> _i<sub>pα</sub>_ = critical gradient for BEP progression (adjusted for pipe path inclination)
> _i<sub>avf</sub>_ = average horizontal gradient in the foundation at the pipe head

For deterministic analysis, the FS is calculated using the most likely values of the random variables and summarized in a table. Cells that do not apply have
a gray background. For probabilistic analysis, the FS is calculated as described for the deterministic analysis but for the mean values of the random variables,
and multiple iterations are performed by sampling the distributions in step 2. The probability of BEP progression is equal to the percentage of iterations that
resulted in a FS less than 1 [(P(FS < 1)]. For probabilistic analysis performed without using @RISK, 1,000 iterations are used. For probabilistic analysis using
@RISK, the number of iterations is user-specified, and “@RISK” displays in parentheses after the number of iterations for this scenario. If cycling through
iterations using @RISK, the displayed results are no longer mean values; they are the selected iteration’s values. For deterministic and probabilistic analyses,
cells with FS less than 1 have an orange background. <FigReference parentDocId="108" figKey="schmertmann-deterministic-output"/> illustrates the deterministic
tabular output, and <FigReference parentDocId="108" figKey="schmertmann-probabilistic-output"/> illustrates the probabilistic tabular output without using @RISK.
