# Site Class Effects

The Site Class Effects worksheet modifies the ground motions from legacy datasets for site class effects because they are only available for NEHRP
site class boundary B/C.

## Site Class

In step 1, use the drop-down list to select the NEHRP site class according to ASCE/SEI 7-16 <Citation citationKey="ASCE2017" /> as <FigReference figKey="figure-17" /> illustrates. The
RMC Site Classification Toolbox, part of the RMC Seismic Hazard Suite, helps assess the site class. <TableReference tableKey="table-2" /> provides
the available site classes for the general procedures for site class adjustment in ASCE/SEI 7-16 used to modify the ground motions for site class
effects. Ensure the selected NEHRP site class is consistent with the value selected on Modern NSHM worksheet.

## Adjusted Ground Motions

In step 2, the general procedures in ASCE/SEI 7-16 <Citation citationKey="ASCE2017" /> are used to adjust the NEHRP site class boundary B/C hazard curves for the user-specified site
class.

If a short-term NSHM is selected for the model on the Legacy NSHM worksheet, the return period to truncate the short-term hazard curves is input as
{"\n"}<FigReference figKey="figure-18" /> illustrates. Otherwise, conditional formatting applies a gray background to this input. The short-term
(one-year) forecasts must not be applied for very low rates of exceedance because hazard estimates from induced earthquakes are not compatible with
estimates of long-term seismic hazard caused by tectonic processes. Therefore, the mean seismic hazard curves are truncated at this user-specified return
period.

<Figure
  figKey="figure-18"
  src="figures/toolbox-technical-manuals/seismic-hazard-suite/seismic-hazard-curves/v1.0/figures/figure18.png"
  alt="Step 2 of Site Class Effects worksheet: Return period input for short-term NSHM."
  caption="Step 2 of Site Class Effects worksheet: Return period input for short-term NSHM."
/>

The general procedures in ASCE-SEI 7-16 are used to adjust the NEHRP site class boundary B/C hazard curves for USGS legacy datasets. The ground motions for PGA,
0.2-second SA (S<sub>S</sub>), and 1-second SA (S<sub>1</sub>) are multiplied by the site coefficients in ASCE/SEI 7-16 <Citation citationKey="ASCE2017" /> to adjust for the user-specified site
class to obtain the modified PGA (PGA<sub>M</sub>), 0.2-second SA (S<sub>MS</sub>), and 1-second SA (S<sub>M1</sub>) used on the Hazard Curves worksheet, as shown
in <EquationReference equationKey="equation-2" />, <EquationReference equationKey="equation-3" />, and <EquationReference equationKey="equation-4" />.

<Equation equationKey="equation-2" equation="PGA_{M} = F_{PGA}PGA" />

<Equation equationKey="equation-3" equation="S_{MS} = F_{a}S_{S}" />

<Equation equationKey="equation-4" equation="S_{M1} = F_{v}S_{1}" />

where:

> F<sub>PGA</sub> = PGA site coefficient  
> F<sub>a</sub> = short-period site coefficient  
> F<sub>v</sub> = long-period (1-second SA) site coefficient

The coefficients to adjust for site class effects in ASCE/SEI 7-16 <Citation citationKey="ASCE2017" /> depend on the ground motion levels for soil sites because of non-linear response at
higher ground motions. Therefore, site class coefficients are linearly interpolated for intermediate ground motion levels as indicated in ASCE/SEI
7-16. <TableReference tableKey="table-3" /> through <TableReference tableKey="table-5" /> provide the site coefficients from ASCE/SEI 7-16 <Citation citationKey="ASCE2017" />.
