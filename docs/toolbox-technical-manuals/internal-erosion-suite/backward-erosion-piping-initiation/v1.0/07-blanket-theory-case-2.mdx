# Blanket Theory Case 2

Case 2 has an impervious top stratum on both the riverside and landside of the levee. The pervious substratum is divided into three zones to apply the
method of fragments as shown in <FigReference figKey="figure-22" />.

## Method of Analysis

The method of analysis is the same as in Case 1.

## Levee Geometry

Step 2 characterizes the levee geometry. The input includes the levee crest elevation, landside toe elevation, distance from the riverside levee toe
to the river (<em>L<sub>1</sub></em>), base width of the levee (<em>L<sub>2</sub></em>), and length of the foundation and top stratum beyond the
landside levee toe (<em>L<sub>3</sub></em>), as illustrated in <FigReference figKey="figure-22" />.

## Pervious Substratum Characterization

The pervious stratum characterization is the same as in Case 1.

## Landside Blanket (Top Stratum) Characterization

Step 4 characterizes the landside (top stratum). The selections in step 1 affect the input for step 4, and cells that do not apply have a gray
background. These cells are not used in subsequent calculations even if data is present. The input includes thickness (<em>z<sub>bl</sub></em>) and
saturated unit weight (<em>&gamma;<sub>sat</sub></em>) of the landside blanket. For deterministic analysis, input only the most likely values. The mean
values used for subsequent calculations are the most likely (or mode) values.

The critical vertical seepage exit gradient for heave/blowout (<em>i<sub>cv</sub></em>), as defined in Terzaghi, Peck, and Mesri (1996), is calculated
using <EquationReference equationKey="equation-6" />.

<Equation
  equationKey="equation-6"
  equation="i_{cv} = \frac{\gamma '}{\gamma_{w}} = \frac{\gamma_{sat}-\gamma_{w}}{\gamma_{w}}"
/>

where:

- &gamma;' = submerged unit weight of the landside blanket
- &gamma;<sub>sat</sub> = saturated unit weight of the landside blanket
- &gamma;<sub>w</sub> = unit weight of water

<FigReference figKey="figure-23" /> illustrates the deterministic input.

  figKey="figure-23"

For probabilistic analysis without using @RISK, input the minimum and maximum values in addition to the most likely value, and triangular
distributions represent the random variables. The mean values used in subsequent calculations are the average of the minimum, most likely, and maximum
values. The triangular distribution for <em>i<sub>cv</sub></em> is calculated using the minimum, most likely, and maximum values for

<em>
  &gamma;<sub>sat</sub>
</em>
. Since
<em>
  i<sub>cv</sub>
</em>
<sub> </sub>is a function of the random variable
<em>
  &gamma;<sub>sat</sub>
</em>
, it is correlated with
<em>
  &gamma;<sub>sat</sub>
</em>
, and is also a random variable. <FigReference figKey="figure-24" /> is an example of the
probabilistic input without using @RISK.

in the fourth column. The mean values used for subsequent calculations are the mean for the @RISK distribution in the third column.

they are the selected iteration’s values.

## Blanket Theory Assumptions

Step 5 checks the BT (method of fragments) assumptions against the input parameters to ensure essentially vertical equipotential lines. For
deterministic analysis, the assumptions are checked for the most likely values of the random variables. For probabilistic analysis, the assumptions
are checked for the mean values of the random variables. Values outside of the model assumptions have an orange background.
{"\n"}<FigReference figKey="figure-26" /> illustrates the check of BT assumptions.

## Seepage Characterization

Step 6 calculates the net hydraulic head on the levee (<em>H</em>) is calculated the same as in Case 1. The flow or seepage per unit length of the
levee (<em>Q<sub>S</sub></em>) is calculated using <EquationReference equationKey="equation-7" />.

Please replace the following component with your KaTeX-formatted equation:

<Equation equationKey="equation-7" equation="Q_{S} = k_{h,f}H\frac{d}{L_{1}+L_{2}+L_{3}}" />

where:

- <em>
    k<sub>h,f</sub>
  </em>
  = horizontal permeability of the pervious substratum
- <em>H</em> = net hydraulic head on the levee
- <em>
    L<sub>1</sub>
  </em>
  = distance from the riverside levee toe to the river
- <em>
    L<sub>2</sub>
  </em>
  = base width of the levee
- <em>
    L<sub>3</sub>
  </em>
  = length of the foundation and top stratum beyond the landside levee toe
- <em>d</em> = thickness of the pervious substratum

The format for the tabular output is the same as in Case 1.

## Likelihood of Heave/Blowout at Landside Toe

Step 7 calculates the excess hydraulic head at the landside levee toe (<em>h<sub>o</sub></em>) using <EquationReference equationKey="equation-8" />.

<Equation equationKey="equation-8" equation="h_{o} = H\frac{L_{3}}{L_{1}+L_{2}+L_{3}}" />

where:

- <em>H</em> = net hydraulic head on the levee
- <em>
    L<sub>1</sub>
  </em>
  = distance from riverside levee toe to the river
- <em>
    L<sub>2</sub>
  </em>
  = base width of the levee
- <em>
    L<sub>3</sub>
  </em>
  = length of the foundation and top stratum beyond the landside levee toe

The vertical seepage exit gradient at the landside levee toe (<em>i<sub>v</sub></em>) is calculated using
{"\n"}<EquationReference equationKey="equation-9" />.

<Equation equationKey="equation-9" equation="i_{v} = \frac{h_o}{z_{bl}}" />

where:

- <em>
    h<sub>o</sub>
  </em>
  = excess hydraulic head at the landside levee toe
- <em>
    z<sub>bl</sub>
  </em>
  = thickness of the landside blanket (top stratum)

The factor of safety against heave/blowout (based on vertical seepage gradients) at the landside levee toe (<em>FS<sub>vg</sub></em>) is calculated
using <EquationReference equationKey="equation-10" />.

<Equation equationKey="equation-10" equation="FS_{vg} = \frac{i_{cv}}{i_v}" />

where:

- <em>
    i<sub>cv</sub>
  </em>
  = critical vertical seepage exit gradient for heave/blowout
- <em>
    i<sub>v</sub>
  </em>
  = vertical seepage exit gradient at the landside levee toe

For deterministic analysis, <em>FS<sub>vg</sub></em> is calculated for each headwater level using the most likely values of the random variables and
is summarized in a table. Cells that do not apply have a gray background. For probabilistic analysis, <em>FS<sub>vg</sub></em> is calculated as
described for the deterministic analysis but for the mean values of the random variables, and multiple iterations are performed by sampling the
distributions in steps 3 and 4. The probability of heave/blowout is equal to the percentage of iterations that resulted in a <em>FS<sub>vg</sub></em>
less than 1 [P(<em>FS<sub>vg</sub></em> < 1)]. For probabilistic analysis performed without using @RISK, 1,000 iterations are used. For probabilistic
analysis using @RISK, the number of iterations is user-specified, and “@RISK” displays in parentheses after the number of iterations for this
scenario. If cycling through iterations using @RISK, the displayed results are no longer mean values; they are the selected iteration’s values. For
deterministic and probabilistic analyses, cells with <em>FS<sub>vg</sub></em> less than 1 have an orange background.
{"\n"}<FigReference figKey="figure-27" /> illustrates the deterministic tabular output, and <FigReference figKey="figure-28" /> illustrates the
probabilistic tabular output without using @RISK.

## Likelihood of Heave/Blowout at Given Distance from Landside Toe

Step 8 calculates the excess hydraulic head at a distance x from the landside levee toe (<em>h<sub>x</sub></em>) using
{"\n"}<EquationReference equationKey="equation-11" />.

<Equation equationKey="equation-11" equation="h_{x} = h_{o}\frac{L_{3}-x}{L_3}" />

where:

- <em>
    h<sub>o</sub>
  </em>
  = excess hydraulic head at the landside levee toe
- <em>
    L<sub>3</sub>
  </em>
  = length of the foundation and top stratum beyond the landside levee toe
- <em>x</em> = distance from the landside levee toe

The vertical seepage exit gradient at a distance <em>x</em> from the landside levee toe (<em>i<sub>v</sub><sub>,x</sub></em>) is calculated using
Equation 12.

<Equation equationKey="equation-12" equation="i_{v,x} = \frac{h_x}{z_{bl}}" />

where:

- <em>
    h<sub>x</sub>
  </em>
  = excess hydraulic head at a distance x from the landside levee toe
- <em>
    z<sub>bl</sub>
  </em>
  = thickness of the landside blanket (top stratum)

The factor of safety against heave/blowout (based on vertical seepage gradients) at a distance <em>x</em> from the landside levee toe
(<em>FS<sub>vg,x</sub></em>) is calculated using <EquationReference equationKey="equation-13" />.

<Equation equationKey="equation-13" equation="FS_{vg,x} = \frac{i_{cv}}{i_{v,x}}" />

where:

- <em>
    i<sub>cv</sub>
  </em>
  = critical vertical seepage exit gradient for heave/blowout
- <em>
    i<sub>v,x</sub>
  </em>
  = vertical seepage exit gradient at a distance x from the landside levee toe

and is summarized in a table. Cells that do not apply have a gray background. For probabilistic analysis, <em>FS<sub>vg,x</sub></em> is calculated as

iterations that resulted in a <em>FS<sub>vg</sub><sub>,x</sub></em> less than 1 [P(<em>FS<sub>vg</sub><sub>,x</sub></em> < 1)]. For probabilistic
analysis performed without using @RISK, 1,000 iterations are used. For probabilistic analysis using @RISK, the number of iterations is user-specified,
and “@RISK” displays in parentheses after the number of iterations for this scenario. If cycling through iterations using @RISK, the displayed
