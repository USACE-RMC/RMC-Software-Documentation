# Merging to Main

## Merge Requirements

Before merging, ensure:

<ProcessList
  items={[
    { title: 'At least 1 approval from an Organization Owner' },

## Merge Process

**GitHub web interface (recommended):**

<ProcessList
  items={[
    { title: 'Click "Merge"' },
    { title: 'Edit the commit message to be clear and descriptive' },
    { title: 'Confirm merge' },
    { title: 'Delete the source branch automatically (check the box)' },
  ]}
  numberStyle="plain"
/>

**Command line (if needed):**

```bash
git checkout main
git pull origin main
git merge --squash feature/your-feature
git commit -m "feat(scope): clear description of feature"
git push origin main
```

## Post-Merge

**Immediately after merging:**

<ProcessList
  items={[
    { title: 'Delete the feature branch (automated via GitHub)' },

<ProcessList
