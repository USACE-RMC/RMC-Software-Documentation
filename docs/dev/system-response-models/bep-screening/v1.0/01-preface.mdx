# Preface

This document provides a technical reference for the Backward Erosion Piping (BEP) screening model implemented in the System Response library. The
model computes a system response probability (SRP) as a function of headwater elevation for the backward erosion piping failure mode. It is designed
specifically for the Dam Screening Tool (DST), where it operates with screening-level simplifications to reduce input burden while preventing false
negatives.

The intended audience includes dam safety engineers evaluating model applicability, peer reviewers checking the technical basis of screening-level
results, and developers who need to understand the model logic before modifying it.
