# Preface

This document provides a technical reference for the Backward Erosion Piping (BEP) screening model implemented in the System Response library. The model computes a system response probability (SRP) as a function of headwater elevation for the backward erosion piping failure mode. It is designed specifically for the Dam Screening Tool (DST), where it operates with screening-level simplifications to reduce input burden while preventing false negatives.

The intended audience includes dam safety engineers evaluating model applicability, peer reviewers checking the technical basis of screening-level results, and developers who need to understand the model logic before modifying it.

## Document Organization

This document is organized into the following chapters:

- **Chapter 2 &mdash; Model Overview** provides a high-level description of what the model computes, when to use it, and how it relates to the full BEP toolbox models.

- **Chapter 3 &mdash; Input Parameters** presents the complete set of input parameters, including hydraulic inputs, geometry, qualitative selections, statistical distributions, and scalar probabilities.

- **Chapter 4 &mdash; Event Tree** describes the seven-node event tree structure used to compute the system response probability, including the engineering question answered by each node and the SRP formula.

- **Chapter 5 &mdash; Condition Routing** documents the six site conditions determined by three Boolean flags, the geometry scenarios associated with each condition, and how probabilities are assigned to the toe and daylighting exit paths.

- **Chapter 6 &mdash; Delegated Models** explains how the screening model constructs and invokes the BT Case 4 initiation and Schmertmann progression sub-models, including the fixed parameters and screening simplifications applied.

- **Chapter 7 &mdash; Assumptions and Limitations** enumerates the screening-level simplifications, their rationale, their effect on results, and the conditions under which they may not hold.

- **Chapter 8 &mdash; Output and Interpretation** describes the result arrays produced by the model, the governing SRP computation, and guidance for interpreting screening-level results.
