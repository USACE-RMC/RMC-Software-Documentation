# Delegated Models

The BEP screening model delegates two computations to existing toolbox models:
[BT Case 4](/docs/toolbox-technical-manuals/internal-erosion-suite/backward-erosion-piping-initiation/v1.0/blanket-theory-case-4) for initiation
analysis and [Schmertmann](/docs/toolbox-technical-manuals/internal-erosion-suite/backward-erosion-piping-progression/v1.0/schmertmann) for
progression analysis. The screening model constructs the configuration for each sub-model internally, applying fixed screening-level parameters. The
sub-models themselves execute identically to their standalone toolbox counterparts &mdash; only the configuration differs.

## BT Case 4 &mdash; Initiation

The BT Case 4 model computes seepage quantities and vertical exit gradients for an embankment with no upstream blanket and an impervious downstream
blanket. The screening model uses BT Case 4 to evaluate two phenomena depending on the site condition:

- **Blowout** (conditions 3 and 5): The probability that uplift pressure beneath the downstream blanket exceeds the blanket's weight, causing rupture
  at the toe. The model output `ToePInit` (fraction of Monte Carlo realizations where factor of safety against vertical gradient is less than 1.0) is
  used as P(unfiltered exit).

- **Heave** (conditions 4 and 6): The probability that the vertical exit gradient at the boil location exceeds the critical gradient of the foundation
  sand. The model output `XPInit` (probability of initiation at distance x) is used as P(initiation).

### BT Case 4 Governing Equations

The BT Case 4 model computes the following quantities for each Monte Carlo realization and headwater level. The model samples foundation thickness
($d$), blanket thickness ($z_{bl}$), and saturated unit weight ($\gamma_{sat}$) from their respective distributions.

**Total seepage length:**

<Equation equationKey="bep-screening-eq-ltotal" equation="L_{\text{total}} = 0.43d + L_2 + L_3" />

where:

> _0.43d_ = effective upstream entrance length for an open seepage entrance  
> _L<sub>2</sub>_ = structure width  
> _L<sub>3</sub>_ = downstream blanket length

**Excess head at the toe:**

<Equation
  equationKey="bep-screening-eq-ho"
  equation="h_o = \begin{cases} H & \text{UseFullHead = true (infinite blanket)} \\ H \times \dfrac{L_3}{L_{\text{total}}} & \text{UseFullHead = false} \end{cases}"
/>

where:

> _H_ = max(HW &minus; TW, 0), the net driving head

When `UseFullHead` is true, the full net head is applied at the toe, modeling an infinite downstream blanket with no head reduction along the seepage
path.

**Critical vertical gradient:**

<Equation equationKey="bep-screening-eq-icv" equation="i_{cv} = \frac{\gamma_{\text{sat}} - \gamma_w}{\gamma_w}" />

where:

> _&gamma;<sub>w</sub>_ = 62.4 pcf, the unit weight of water

**Vertical exit gradient and factor of safety at the toe:**

<Equation equationKey="bep-screening-eq-ivtoe" equation="i_{v,o} = \frac{h_o}{z_{\text{bl}}}, \quad \text{FS}_{vg,o} = \frac{i_{cv}}{i_{v,o}}" />

**Probability of initiation at the toe:**

<Equation
  equationKey="bep-screening-eq-pinittoe"
  equation="P_{\text{init,toe}} = \frac{\text{count}(\text{FS}_{vg,o} < 1.0)}{N_{\text{iterations}}}"
/>

**Head and gradient at distance x** (for heave conditions):

<Equation
  equationKey="bep-screening-eq-hx"
  equation="h_x = \begin{cases} h_o & \text{UseFullHead = true} \\ h_o \times \dfrac{L_3 - x}{L_3} & \text{if } x < L_3 \\ 0 & \text{if } x \geq L_3 \end{cases}"
/>

<Equation
  equationKey="bep-screening-eq-ivx"
  equation="i_{v,x} = \frac{h_x}{z_{\text{bl}}}, \quad \text{FS}_{vg,x} = \frac{i_{cv}}{i_{v,x}}, \quad P_{\text{init,x}} = \frac{\text{count}(\text{FS}_{vg,x} < 1.0)}{N_{\text{iterations}}}"
/>

## Schmertmann &mdash; Progression

The Schmertmann model computes the probability of backward erosion piping advancement at each headwater level by comparing the applied hydraulic
gradient against the critical horizontal gradient through Monte Carlo simulation.

### Exit Geometry (ExitIsHole)

The `ExitIsHole` parameter controls the three-dimensional exit correction factor (C<sub>3d</sub>) in the Schmertmann model:

- **ExitIsHole = true** (C<sub>3d</sub> = 0.5): Applied to the toe path when a confining layer is present. A hole exit represents a concentrated
  seepage point where flow converges through a breach in the confining layer. Van Beek et al. (2015) found that three-dimensional flow configurations
  toward a single point resulted in critical gradients approximately half those predicted by the two-dimensional model.

- **ExitIsHole = false** (C<sub>3d</sub> = 1.0): Applied to the daylighting exit path and to the toe path when no confining layer is present. A plane
  exit represents seepage along an exposed surface where flow is approximately two-dimensional.

The C<sub>3d</sub> factor is incorporated into the overall field correction factor:

<Equation equationKey="bep-screening-eq-cof" equation="C_{OF} = C_{FC} \times C_{3d}" />

where:

> _C<sub>FC</sub>_ = fines content correction factor

The critical horizontal gradient is then:

<Equation equationKey="bep-screening-eq-ich" equation="i_{ch} = i_{p\alpha} \times C_{OF}" />

Because the toe path (with confining layer) uses C<sub>3d</sub> = 0.5, the critical gradient is halved relative to the daylight path. This means the
toe path will generally produce a higher probability of progression, making it the more critical path when a confining layer is present.

### Schmertmann Critical Gradient

The Schmertmann model computes the critical horizontal gradient as the product of the base gradient from laboratory piping tests and a series of
correction factors:

<Equation
  equationKey="bep-screening-eq-ich-full"
  equation="i_{ch} = \frac{i_{\text{pmt}} \times C_D \times C_L \times C_S \times C_K \times C_\gamma \times C_Z}{C_R} \times C_\alpha \times C_{OF}"
/>

In the screening model, several correction factors are fixed:

- $C_K = 1.0$ (anisotropy disabled, $R_{kf} = 1.0$)
- $C_Z = 1.0$ (no underlayer)
- $C_\alpha = 1.0$ (horizontal pipe path, $\alpha = 0$)
- $C_R = 1.0$ (reference baseline)

The remaining correction factors ($C_D$, $C_L$, $C_S$, $C_\gamma$, $C_{OF}$) are computed from the sampled parameter values in each Monte Carlo
iteration. The probability of progression at each headwater level is the fraction of iterations where the applied gradient exceeds the critical
gradient:

<Equation equationKey="bep-screening-eq-iadv" equation="i_{\text{adv}} = \frac{\Delta H}{L}, \quad P_{\text{adv}} = P(i_{\text{adv}} > i_{ch})" />

where:

> _&Delta;H_ = HW &minus; TW, the net driving head  
> _L_ = seepage path length

### I<sub>pmt</sub> Quantile Entries

The base critical gradient (i<sub>pmt</sub>) is derived from laboratory piping model test data. The Schmertmann model uses a quantile regression
approach where the gradient at each quantile is computed as a linear function of the coefficient of uniformity:

<Equation equationKey="bep-screening-eq-ipmt" equation="i_{\text{pmt}}(q) = \text{slope}(q) \times C_u + \text{intercept}(q)" />

The screening model constructs the `IPmtEntries` list from the static arrays in `SchmertmannData` (`GradientQuantiles`, `GradientQuantileSlopes`,
`GradientQuantileIntercepts`). These entries define the cumulative distribution of the base gradient as a function of soil gradation.
