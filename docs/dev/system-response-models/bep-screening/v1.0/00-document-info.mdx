reportAbstract: Technical reference documenting the Backward Erosion Piping (BEP) screening model in the System Response library. The model computes a system response probability (SRP) for backward erosion piping at each headwater level using a seven-node event tree. It integrates condition-based routing logic with delegated BT Case 4 initiation and Schmertmann progression sub-models, applying screening-level simplifications appropriate for the Dam Screening Tool.
