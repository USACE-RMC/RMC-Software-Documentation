# Input Parameters

All input parameters are defined in the `BEPScreeningConfig` class. The parameters are organized into five groups: hydraulic inputs, dam geometry,
BEP-specific selections, statistical distributions, and scalar probabilities.

## Hydraulic Inputs

The model requires paired headwater and tailwater elevation series that define the hydraulic loading across the range of pool levels to be evaluated.

The headwater and tailwater knots are expanded internally to 50 evenly spaced headwater steps using linear interpolation
(`HydraulicUtils.ExpandHWSeries`). This expansion produces the headwater array used for all subsequent calculations.

The total structure width used in sub-model calculations is computed as:

<Equation equationKey="bep-screening-eq-l2" equation="L_2 = \text{EmbankmentBaseWidth} + \text{BermLength}" />

The berm is included in L<sub>2</sub> (structure width) rather than L<sub>3</sub> (blanket length) because it is part of the water-retaining structure
and contributes additional overburden weight against uplift pressures.
