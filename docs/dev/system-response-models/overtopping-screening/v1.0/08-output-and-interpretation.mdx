# Output and Interpretation

The overtopping erosion screening model produces an `OvertoppingResult` object containing four system response curves, grass removal diagnostic
counts, and computation time metadata.

## Interpreting the System Response Curve

### Reading the Curve

The system response curve (`ResponseFunction`) maps each overtopping depth to a breach probability between 0 and 1. The curve is expected to be
monotonically non-decreasing: larger overtopping depths produce larger discharge, longer durations, and higher shear stresses, all of which increase
the probability of breach.

Key characteristics of a typical system response curve:

- **Low depths (< 0.5 ft):** Breach probability is typically near zero. The overtopping discharge is small, durations are short, and the grass cover
  (if present) is likely sufficient to prevent soil exposure.
- **Moderate depths (0.5 – 2.0 ft):** Breach probability begins to increase. The rate of increase depends strongly on the erodibility classification
  and grass cover quality.
- **Large depths (> 2.0 ft):** Breach probability approaches 1.0 for erodible materials. Resistant and very resistant materials may maintain
  probabilities below 1.0 even at large overtopping depths.

This decomposition helps identify whether the grass cover or the soil erodibility dominates the system response at a given depth. If the grass removal
probability is high but the conditional breach probability is low, the embankment material is resistant enough to survive even after grass failure. If
the conditional breach probability is high but grass removal probability is low, the grass cover is the primary defense against breach.

## Screening vs. Detailed Analysis

The screening system response curve should be interpreted as an estimate intended to identify structures that warrant further analysis. The model has
been validated against Quantitative Risk Assessment (QRA) response functions from multiple USACE dam safety studies (Benbrook, Cherry Creek,
Delaware, Grapevine, J. Percy Priest, Keystone, Kinzua, Lookout Point, Mojave, Painted Rock, Stillhouse Hollow, W. Kerr Scott, and Wister Dams).

A high screening breach probability does not necessarily indicate a high risk of failure — it indicates that the structure should be evaluated with
detailed overtopping erosion analysis software where:

- Site-specific hydrographs from hydrologic routing can be used instead of the gamma-shaped approximation.
- Zoned embankment geometry can be represented with distinct material zones and internal features.
- Calibrated erosion parameters from laboratory testing (jet erosion tests, hole erosion tests) can replace the default category distributions.
- Headcut erosion mechanics can be modeled explicitly.
- Riprap and other armoring systems can be accounted for.

A low screening breach probability provides reasonable confidence that overtopping erosion is not a critical concern for the structure at the screening
level.
