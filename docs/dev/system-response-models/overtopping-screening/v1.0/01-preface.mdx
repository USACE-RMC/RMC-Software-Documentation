# Preface

This document provides a technical reference for the overtopping erosion screening model implemented in the System Response library. The model
computes a system response curve relating overtopping depth to the probability of embankment breach due to erosion. It is designed specifically for
the Dam Screening Tool (DST), where it operates with screening-level simplifications to provide rapid, conservative estimates of overtopping erosion
risk for embankment dams.

The intended audience includes dam safety engineers evaluating model applicability, peer reviewers checking the technical basis of screening-level
results, and developers who need to understand the model logic before modifying it.

## Document Organization

This document is organized into the following chapters:

- **Chapter 2 — Model Overview** provides the purpose of the model, its intended use within the Dam Screening Tool, and a high-level summary of the
  computational algorithm.

- **Chapter 3 — Input Parameters** documents all configuration parameters organized by category, including geometry, material classification,
  hydrograph settings, hydraulic parameters, correlation coefficients, and Monte Carlo settings.

- **Chapter 4 — Hydrograph Generation** describes the gamma-shaped hydrograph used to model time-varying overtopping discharge, including the
  duration scaling relationship and the peak discharge computation.

- **Chapter 5 — Erosion Model** details the two-phase erosion simulation: grass cover removal based on cumulative excess discharge, followed by
  progressive embankment erosion using the excess shear stress equation.

- **Chapter 6 — Monte Carlo Simulation** explains the correlated material property sampling via Gaussian copula, the parallel simulation framework,
  and the assembly of system response curves from Monte Carlo results.

- **Chapter 7 — Assumptions and Limitations** documents each simplifying assumption with its rationale, effect on results, and the conditions under
  which it may not hold.

- **Chapter 8 — Output and Interpretation** describes the result structure, how to interpret the system response curve, and the relationship between
  screening results and detailed analysis.
