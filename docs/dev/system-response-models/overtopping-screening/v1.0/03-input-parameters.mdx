# Input Parameters

All input parameters are defined in the `OvertoppingConfig` class. The parameters are organized into seven groups: unit system, embankment geometry,
material classification, hydrograph settings, hydraulic parameters, correlation coefficients, and Monte Carlo settings.

## Unit System

The model supports both US Customary and SI unit systems. The unit system selection affects default material parameters, physical constants, and
dimensional quantities throughout the analysis.

(dimensionless). For SI units, $K_d$ is converted to (m/hr)/Pa and $\tau_c$ is converted to Pa using the conversion factors in `PhysicalConstants`.

The critical velocity ($u_c$) is the flow velocity threshold above which grass damage initiates. The erosional limit ($G_f$) is the cumulative work

unit system — the model performs internal unit conversions as needed.

## Correlation Coefficients

The model uses a Gaussian copula to generate correlated samples of the three material properties. Two correlations are specified directly; the third
is implied.

<TableVertical
  tableKey="overtopping-screening-table-correlation"
  alt="Correlation coefficient parameters"
  caption="Correlation coefficient parameters."
  colWidths={[18, 18, 8, 10, 12, 'minmax(18ch, 1fr)']}
  headers={[
    [{ value: 'Parameter' }, { value: 'Code Variable' }, { value: 'Type' }, { value: 'Default' }, { value: 'Valid Range' }, { value: 'Description' }],
  ]}
  columns={[
    ['Correlation Kd–τc', 'Correlation τc–n'],
    ['CorrelationKdTauC', 'CorrelationTauCN'],
    ['double', 'double'],
    ['-0.6', '-0.2'],
    ['-1 to 1', '-1 to 1'],
    [
      'Correlation between the erodibility coefficient (Kd) and critical shear stress (τc). A negative value indicates that more erodible soils (higher Kd) tend to have lower critical shear stress (lower τc), which is consistent with field observations.',
      'Correlation between critical shear stress (τc) and Manning\'s roughness (n). A negative value indicates that soils with higher critical shear stress tend to have smoother surfaces.',
    ],
  ]}
/>

The implied correlation between $K_d$ and $n$ is computed as the product of the two specified correlations:
$\rho(K_d, n) = \rho(K_d, \tau_c) \times \rho(\tau_c, n)$. This relationship arises because the correlations between $K_d$ and $\tau_c$ and between
$\tau_c$ and $n$ are established through data analysis, which implies a transitive correlation between $K_d$ and $n$. With the default values, the
implied correlation is $(-0.6) \times (-0.2) = 0.12$.
