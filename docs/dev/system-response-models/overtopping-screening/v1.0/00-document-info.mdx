reportAbstract: Technical reference documenting the overtopping erosion screening model in the System Response library. The model computes a system response curve relating overtopping depth to breach probability for embankment dams using physics-based Monte Carlo simulation. It integrates gamma-shaped hydrograph generation, time-varying grass cover removal, and excess shear stress erosion with correlated material property sampling via Gaussian copula. The model is designed for the Dam Screening Tool as a screening-level assessment of overtopping erosion risk.
