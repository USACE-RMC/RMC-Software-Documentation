# Model Overview

## Purpose

The overtopping erosion screening model computes a system response curve for the overtopping erosion failure mode as a function of overtopping depth.
The system response probability (SRP) at each depth represents the conditional probability of embankment breach due to erosion, given that the
reservoir overtops the dam crest by that amount. The model produces a curve — an array of SRP values across a range of overtopping depths — that
characterizes the embankment's vulnerability to breach from overtopping flow.

## Intended Use

The model is designed specifically for the Dam Screening Tool (DST), where it serves as a screening-level assessment of overtopping erosion risk. It
makes simplifying assumptions with the goal of:

- **Providing rapid estimates** — The model runs a physics-based Monte Carlo simulation in seconds, enabling batch processing of many structures
  without manual hydrograph development or detailed site characterization.
- **Reducing input burden** — The model requires only embankment geometry, material classification, grass cover quality, and overtopping duration.
  Material properties are drawn from default distributions based on the selected erodibility classification.

The model is not intended for detailed or semi-quantitative risk assessments. For higher-level analyses, engineers should use more detailed
overtopping erosion analysis software such as WinDAM C<Citation citationKey="WinDAM" />, DLBreach, or EMBREA, which allow for site-specific
hydrograph inputs, zoned embankment geometry, and calibrated erosion parameters.

## Physics Basis

The model combines several established engineering methods:

- **Broad-crested weir equation** for computing overtopping discharge from depth<Citation citationKey="Fell2015" />
- **Gamma-shaped hydrograph** for representing the time history of overtopping flow
- **CIRIA grass cover erosion resistance** for modeling protective vegetation removal under sustained overtopping<Citation citationKey="Hughes2015" /><Citation citationKey="Thornton2014" />
- **Excess shear stress erosion equation** for computing headcut-driven embankment erosion, based on the approach used in WinDAM<Citation citationKey="WinDAM" /><Citation citationKey="Hanson2011" />
- **Briaud (2008) erosion categories** for default material property distributions<Citation citationKey="Briaud2019" /><Citation citationKey="Wibowo2018" />
