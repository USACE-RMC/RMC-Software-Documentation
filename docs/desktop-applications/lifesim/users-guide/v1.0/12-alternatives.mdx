# Alternatives

An <strong>alternative</strong> in LifeSim allows the user to define what input data sources and parameters will make up the simulation computations.
Any combination of input data sources can be used to define an alternative. For example, one alternative may use a road network that allows for
contra-flow and another may use a different hydraulic event or different warning issuance times. Setting up multiple alternatives allows the user to
quantify and compare the effects of operational changes on the hazard analysis. 

Alternatives in LifeSim represent the parameters that define a
simulation and include defining:

- Input data sources (e.g., hydraulics, structure inventory, road network, ECAM data),

- Warning issuance times,

- Destination locations by Emergency Planning Zone (EPZ), and

- Various other parameters (e.g., evacuation parameters, life loss probability, stability criteria).

## Create an Alternative

To create an alternative in LifeSim the user will provide a name, define the input data sources, and provide additional information (e.g., evacuation
parameters).

### Compute Options for an Alternative

LifeSim can analyze a variety of flood damages and impacts through a variety of computational methods. Flood damages and impacts as well as the
methods used to estimate the impacts are selected as part of an alternative definition. For example, when simulating traffic and life loss, users
must define warning issuance scenarios, destination assignments, evacuation parameters, life loss probability functions, and stability threshold
functions. The following sections describe the steps for defining the compute options for an alternative individually for each tab in the
<em>Create New Alternative</em> dialog box (<FigReference figKey="create-new-alt" />).

**NOTE:** for the purposes of the discussion the units listed in this section will be in **US Customary** units (feet). However, the <em>Create New 
Alternative</em> dialog box (<FigReference figKey="create-new-alt" />) allows users to select either <strong>US 
Customary</strong> (feet) <FigureInline src="figures/desktop-applications/lifesim/users-guide/v1.0/inline-images/inline-toggle.png"/> or <strong>System 
International (SI)</strong> (meters).

#### Destination Assignments

When simulating traffic (<strong>Simulate Traffic</strong> <FigureInline src="figures/desktop-applications/lifesim/users-guide/v1.0/inline-images/inline-checkbox.png"/> checkbox 
is selected such as in <FigReference figKey="figure-169" />), the user needs to define the
destinations for the emergency planning zones; from the <em>Create New Alternative</em> dialog box,
click the <strong>Destination Assignments</strong> tab (<FigReference figKey="figure-172" />).

<Figure
  figKey="figure-172"
  src="figures/desktop-applications/lifesim/users-guide/v1.0/figures/figure172.png"
  alt="Create New Alternative – Destination Assignments Tab"
  caption="Create New Alternative – Destination Assignments Tab"
/>

The <strong>Destination Assignments</strong> tab (<FigReference figKey="figure-172" />) is divided into two panels. The left panel
lists all of the emergency planning zones in the selected EPZ dataset (e.g., <strong>Example_EPZ</strong> in <FigReference figKey="figure-172" />).
Therefore, if a selected EPZ dataset has multiple zones, then the list will contain the names of the multiple 
zones (e.g., <strong>Cloverdate</strong> zone, <strong>Geyserville</strong> zone, and so on in <FigReference figKey="figure-172" />). The right panel provides a list
of all of the destination names (e.g., <strong>I-405 South</strong>) in the destination’s 
dataset (e.g., <strong>Ex_Destinations</strong> in <FigReference figKey="figure-172" />).

The user can define unique (for multiple zones when present) destinations for selected EPZs. For example, if the warning message content for a
particular EPZ dictates specific destination locations, those locations can be defined and people within the EPZ will attempt to evacuate to one of the
specified destination locations.

#### Evacuation Parameters

When simulating life loss, the <strong>Evacuation Parameters</strong> tab (<FigReference figKey="figure-173" />) allows the user to
adjust the default traffic evacuation simulation parameters in LifeSim.

The <strong>Evacuation Parameters</strong> tab contains two panels.

<strong>*Evacuation Parameters* left panel</strong>: contains a list of traffic
simulation parameters with defaults (<TableReference tableKey="table-5" />). All available evacuation parameters are set with defaults, but the user
can adjust the parameters by modifying the default values.

<strong>*Depth Driven Willingness to Enter Flooded Road* right panel</strong>: users can define probability density functions for low
clearance (cars) and high clearance (trucks) vehicles willingness to enter (ford) flooded roads. During the evacuation simulation, vehicles
encountering a flooded road will either attempt to ford the road or will attempt to turn around depending on the defined willingness to enter
functions. The *LifeSim Version 2.0 Technical Reference Manual, Chapter 6* provides extensive detail regarding evacuation parameters and depth driven
willingness to enter flooded roads.

low and high hazard. The *LifeSim Version 2.0 Technical Reference Manual, Chapter 9, Section 9.2* provides a detailed explanation regarding the
development of the two new zones and the fatality rate functions for the new zones.

#### Stability Criteria

The <strong>Create New Alternative</strong> dialog box (<FigReference figKey="figure-165" />), <strong>Stability Criteria</strong> tab (
{"\n"}<FigReference figKey="figure-174" />) allows users to adjust the default stability criteria for three groups: human stability and low and high
clearance vehicles. The user can adjust the default stability criteria thresholds (depth, velocity, and functional) for the three groups. The LifeSim
Version 2.0 Technical Reference Manual provides more information on the human and vehicle stability criteria.

<Figure
  figKey="figure-177"
  src="figures/desktop-applications/lifesim/users-guide/v1.0/figures/figure177.png"
  alt="Depth Driven Willingness To Enter Flooded Road Panel – Example Distribution Selected"
  caption="Depth Driven Willingness To Enter Flooded Road Panel – Example Distribution Selected"

To modify the default functions:

1. From the <strong>Create New Alternative</strong> dialog box (<FigReference figKey="figure-165" />), <strong>Stability Criteria</strong> tab (
{"\n"}<FigReference figKey="figure-174" />). From the <strong>Stability Criteria</strong> dropdown list, select the desired group (e.g., <strong>Low
Clearance Vehicles</strong> in <FigReference figKey="figure-174" />).

1. The user can modify the uncertainty about the three thresholds for the selected stability criteria group. From the <strong>Uncertainty
Type</strong> dropdown menu, select the desired uncertainty type and define the uncertainty parameters for the threshold. Modify the listed stability
criteria parameters (<FigReference figKey="figure-174" />) by entering values into the fields manually.

1. The <strong>Functional</strong> <strong>Threshold</strong> has an additional parameter for selecting the function type, from the
<strong>Function</strong> <strong>Type</strong> list (<FigReference figKey="figure-174" />), as either <strong>Depth&#42;Velocity</strong> or
<strong>Depth&#42;(Velocity Squared)</strong>.

1. When done evaluating/modifying the stability criteria parameters for all three groups (humans, low and high clearance vehicles), either click
<strong>OK</strong> to save the alternative and close the <strong>Create New Alternative</strong> dialog box (<FigReference figKey="figure-167" />)
or select the next desired tab and continue to define the new alternative.

## Copy an Alternative

To copy an alternative, from the LifeSim main window (<FigReference figKey="figure-164" />), from the <strong>Study</strong> tab, from the
<strong>Study Tree</strong>, from the <strong>Alternatives </strong>folder, right-click on an alternative name. From the shortcut menu (
{"\n"}<FigReference figKey="figure-176" />), click <strong>Copy</strong>. Another way is from the LifeSim main window, from the <strong>Study</strong>
 menu, point to <strong>Alternatives</strong>, point to <strong>Copy</strong>, click on an alternative name.

Either way, the <strong>Name of New Study Item</strong> dialog box (<FigReference figKey="figure-86" />) opens. Enter a new name in the
<strong>Name</strong> box. Click <strong>OK</strong>, the <strong>Name of New Study Item</strong> dialog box (<FigReference figKey="figure-86" />)
closes and the copied alternative is added to the <strong>Study Tree</strong> under the <strong>Alternatives</strong> folder.

## Delete an Alternative

To delete an alternative, from the LifeSim main window (<FigReference figKey="figure-164" />), from the <strong>Study</strong> tab, from the
<strong>Study Tree</strong>, from the <strong>Alternatives</strong> folder, right-click on an alternative name. From the shortcut menu (
{"\n"}<FigReference figKey="figure-176" />), click <strong>Delete</strong>. Another way is from the LifeSim main window, from the
<strong>Study</strong> menu, point to <strong>Alternatives</strong>, point to <strong>Delete</strong>, click on an alternative name. Either way a
<strong>Confirm Delete</strong> window (<FigReference figKey="figure-177" />) opens asking for confirmation to delete the selected alternative. Click
 <strong>Yes</strong> to confirm the deletion. The <strong>Confirm Delete</strong> window (<FigReference figKey="figure-177" />) closes, and the
selected alternative is deleted, and no longer displays in the <strong>Study Tree</strong> under the <strong>Alternatives</strong> folder.
