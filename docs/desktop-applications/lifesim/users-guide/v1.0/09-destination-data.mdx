# Destination Data

<strong>Destination</strong> <strong>data</strong> is required in a LifeSim study to represent possible evacuation locations during flooding events.
The destination can either be a location within the flooded area that is considered a safe haven for evacuees or locations outside the flooding
extents. The local Emergency Management Agency (EMA) can be a helpful resource for locating safe havens and for information on defining these types of
 locations. The Federal Emergency Management Agency (FEMA) provides contact information for EMAs ().

The destination data is represented as points in space, LifeSim will find the nearest road from a destination point and consider that the evacuation
location. The destinations do not have to be directly on a road; however, only one destination per road segment is allowed. A method to create
destinations and define their location is detailed in [Create Destination Locations
Example](./18-appendix-map-preferences.mdx#create-destination-locations-example).

## Create Destination Datasets

To create a destination dataset from a shapefile:

1. From the LifeSim main window (<FigReference figKey="figure-138" />), from the <strong>Study</strong> tab, from the <strong>Study Tree</strong>,
right-click on <strong>Destinations</strong>, from the shortcut menu, click <strong>Import Destinations From Shapefile</strong>. Another way is from
the LifeSim main window, from the <strong>Study</strong> menu, point to <strong>Destinations Data</strong> (<FigReference figKey="figure-139" />),
click <strong>Import From Shapefile</strong>.

1. Either way, the <strong>Import Destinations From Point Shapefile</strong> dialog box (<FigReference figKey="figure-140" />) will open.

1. Enter a name for the destination dataset in the <strong>Name</strong> box (<FigReference figKey="figure-140" />).

1. To select the shapefile that contains information about destinations, to the right of the <strong>Destinations Shapefile</strong> dropdown menu (
{"\n"}<FigReference figKey="figure-140" />), click . An <strong>Open</strong> browser window (<FigReference figKey="figure-141" />) opens; navigate
to the directory where the desired point shapefile is located. Select the correct point shapefile (e.g., <em>&#42;.shp</em>), click <strong>Open</strong>,
 the <strong>Open</strong> browser closes (<FigReference figKey="figure-141" />) and the user will be returned to the<strong> Import Destinations
From Point Shapefile</strong> (<FigReference figKey="figure-140" />). The location and name of the selected point shapefile will display in
the<strong> Destinations Shapefile</strong> dropdown menu (<FigReference figKey="figure-140" />).

1. Alternatively, if the desired shapefile has already been imported as a map layer into the LifeSim study, then from the <strong>Destinations
Shapefile</strong> dropdown menu (<FigReference figKey="figure-140" />), select the desired map layer.

identifies the destinations (e.g., destination names) in the selected shapefile.

From the LifeSim main window (<FigReference figKey="figure-138" />), from the <strong>Study Tree</strong>, under the <strong>Destinations</strong>
folder (<FigReference figKey="figure-139" />), the name of the destination dataset is displayed.

## Edit a Destination Dataset

To copy a destination dataset from an existing dataset, from the LifeSim main window (<FigReference figKey="figure-138" />), from the
<strong>Study</strong> tab, from the <strong>Study Tree</strong>, from the <strong>Destinations</strong> folder, right-click on a destination dataset,
 from the shortcut menu (<FigReference figKey="figure-142" />), click <strong>Edit</strong>. Another way is from the LifeSim main window, from the
<strong>Study</strong> menu, point to <strong>Destinations Data</strong> (<FigReference figKey="figure-139" />), point to <strong>Edit</strong>, and
click a destination dataset.

<Figure
  figKey="figure-142"
  src="figures/desktop-applications/lifesim/users-guide/v1.0/figures/figure142.png"
  alt="LifeSim Study Tree – Destination Dataset – Shortcut Menu"
  caption="LifeSim Study Tree – Destination Dataset – Shortcut Menu"
/>

Either way, the <strong>Import Destinations From Point Shapefile</strong> dialog box (<FigReference figKey="figure-140" />) opens. From the
<strong>Destination Identifier Field</strong> dropdown menu (<FigReference figKey="figure-140" />), select the attribute field that identifies the
destinations (e.g., destination names) in the selected shapefile. Click <strong>OK</strong>, the <strong>Import Destinations From Point
Shapefile</strong> dialog closes (<FigReference figKey="figure-140" />).

## Display a Destination Dataset

To display a destination dataset, from the LifeSim main window (<FigReference figKey="figure-138" />), from the <strong>Study</strong> tab, from the
<strong>Study Tree</strong>, from the <strong>Destinations</strong> folder, right-click on a destination dataset, from the shortcut menu (
{"\n"}<FigReference figKey="figure-142" />), click <strong>Show In Map Window</strong>. The selected destination dataset will display in the map
window of the LifeSim main window.

Map layer attributes can be accessed for each map layer and is provided in a table for the dataset referenced by the map layer. The destinations
dataset must be displayed in the map window before the attribute table can be accessed. To access a specific map layer Attribute table, from the Map
Layers tab, right-click on a map layer of interest, from the shortcut menu, click Open Attribute Table. The Map Layer Attributes dialog box opens;
review [Map Layer Attributes Dialog Box](./18-appendix-map-preferences.mdx#map-layer-attributes-dialog-box) for more information. Editing of the map
layer attributes is detailed in [Editing Map Layer Features and
Attributes](./18-appendix-map-preferences.mdx#editing-map-layer-features-and-attributes).

## Copy a Destination Dataset

To copy a destination dataset from an existing dataset, from the LifeSim main window (<FigReference figKey="figure-138" />), from the
<strong>Study</strong> tab, from the <strong>Study Tree</strong>, from the <strong>Destinations</strong> folder, right-click on a destination dataset,
 from the shortcut menu (<FigReference figKey="figure-142" />), click <strong>Copy</strong>. Another way is from the LifeSim main window, from the
<strong>Study</strong> menu, point to <strong>Destinations Data</strong> (<FigReference figKey="figure-139" />), point to <strong>Copy</strong>, and
click a destination dataset.

Either way, the <strong>Name of New Study Item</strong> dialog box (<FigReference figKey="figure-86" />) opens. Enter a new name in the
<strong>Name</strong> box. Click <strong>OK</strong>, the <strong>Name of New Study Item</strong> dialog box (<FigReference figKey="figure-86" />)
closes and the copied destination dataset displays in the <strong>Study Tree</strong> under the <strong>Destinations</strong> folder.

## Delete a Destination Dataset

To delete a destination dataset from the LifeSim study and study directory files, from the LifeSim main window (<FigReference figKey="figure-138" />
), from the <strong>Study</strong> tab, from the <strong>Study Tree</strong>, from the <strong>Destinations</strong> folder, right-click on a
destination dataset, from the shortcut menu (<FigReference figKey="figure-142" />), click <strong>Delete</strong>. Another way is from the LifeSim
main window, from the <strong>Study</strong> menu, point to <strong>Destinations Data</strong> (<FigReference figKey="figure-139" />), point to
<strong>Delete</strong>, click a destination dataset.

Either way, a <strong>Verify Deletion of Destination Data</strong> message window (<FigReference figKey="figure-143" />) opens asking for
confirmation to delete the selected destination dataset. Click <strong>Yes</strong>, the <strong>Verify Deletion of Destination Data</strong> message
window closes (<FigReference figKey="figure-143" />), and the selected destination dataset is deleted, and no longer displays in the <strong>Study
Tree</strong> under the <strong>Destinations</strong> folder.
