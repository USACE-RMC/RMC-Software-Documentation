# Create a Project

When you open RMC-TotalRisk, it automatically creates a Blank Project file, as shown in <FigReference figKey="figure-49" />. The blank project
is stored in your local temp directory. You may begin working with the blank project file immediately.

<Figure
figKey="figure-49"
src="figures/desktop-applications/rmc-totalrisk/users-guide/v1.0/figure48.png"
alt="The Blank Project file includes folders such as Hazards, Transforms, System Responses, Consequences, and Risk Analyses."
caption="RMC-TotalRisk blank project."
/>

To save changes to the blank project, click the Save button on the tool bar or under the **File** menu. This opens the **Save Project As…** prompt
(<FigReference figKey="figure-50" />). Enter the desired file name and click the save button in the bottom right. Now you are ready to continue
working with RMC-TotalRisk.

<Figure
figKey="figure-50"
src="figures/desktop-applications/rmc-totalrisk/users-guide/v1.0/figure49.png"
alt="The Save Project As… window allows the user to save the project at their desired location. Users can also change the project file name."
caption="Save project as...."
/>

You can also create a new project by clicking **New Project…** under the **File** menu (<FigReference figKey="figure-51" />) or by clicking
the **New Project** button on the tool bar (<FigReference figKey="figure-52" />). The first time you use RMC TotalRisk, the recent projects
list will be empty.

<Figure
figKey="figure-51"
src="figures/desktop-applications/rmc-totalrisk/users-guide/v1.0/figure50.png"
alt="The File menu includes options to create a new project, open an existing project, save a project, and exit the widow."
caption="Create new project from the file menu."
/>

<Figure
figKey="figure-52"
src="figures/desktop-applications/rmc-totalrisk/users-guide/v1.0/figure51.png"
alt="The New Project button on the toolbar creates a Blank Project file with folders such as Hazards, Transforms, System Responses, Consequences, and Risk Analyses."
caption="Create new project from the tool bar."
/>

The project properties are shown in the **Properties** window (<FigReference figKey="figure-53" />), which is typically on the right hand
side of the main window. You may edit the project name and description.

<Figure
figKey="figure-53"
src="figures/desktop-applications/rmc-totalrisk/users-guide/v1.0/figure52.png"
alt="The Properties window shows the name and description of the project. This window also shows when the project was created and last modified. It also shows the project file directory, and the software version."
caption="Project properties."
/>
