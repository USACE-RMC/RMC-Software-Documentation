# Documentation Guide

## Docusaurus Project Setup and Usage

This guide will help you set up and use Docusaurus, a modern static website generator, for the RMC Software Documentation project.

The documentation is organized to help you:

- Set up your development environment
- Understand the project structure
- Build, run, and deploy the documentation site
- Use and extend the provided React components
- Follow best practices for editing and maintaining documentation

> Use the sidebar to navigate through setup, editing, versioning, and advanced usage topics.
